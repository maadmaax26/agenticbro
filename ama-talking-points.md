### X Profile Scanning

**Process:**
1. User enters X username (@username)

3. Extracts real-time data:
   - Follower count
   - Verification status
   - Post count
   - Engagement rate
   - Bio and website
   - Recent tweets
4. Analyzes 10 red flags:
   - Guaranteed returns
   - Private alpha
   - Unrealistic claims

### Risk Scoring Algorithm

**10 Red Flags:**
1. **Guaranteed Returns** (weight: 9/10)
   - Keywords: "guaranteed", "guarantee", "risk-free"
   - Indicates unrealistic promises

2. **Private Alpha** (weight: 9/10)
   - Keywords: "private alpha", "insider", "exclusive access"
   - Common pump-and-dump tactic

3. **Unrealistic Claims** (weight: 9/10)
   - Keywords: "x10", "x100", "x500", "500%", "1000%"
   - Promises unrealistic returns

4. **Urgency Tactics** (weight: 8/10)
   - Keywords: "act now", "limited spots", "act fast", "last chance"
   - Creates FOMO to rush users

5. **No Track Record** (weight: 8/10)
   - Keywords: "no track record", "no pnl", "no performance"
   - New or inactive accounts

6. **Requests Crypto** (weight: 10/10)
   - Keywords: "send crypto", "transfer", "deposit", "investment"
   - Direct request for payment

7. **No Verification** (weight: 5/10)
   - Keywords: "not verified", "unverified", "no blue check"
   - Lacks verification badge

8. **Fake Followers** (weight: 6/10)
   - Keywords: "bot farm", "fake followers", "bought followers"
   - Inauthentic engagement

9. **New Account** (weight: 7/10)
   - Keywords: "joined [date]", "new account", "recently joined"
   - Very recent accounts

10. **VIP Upsell** (weight: 6/10)
    - Keywords: "vip tier", "exclusive access", "membership", "subscribe"
    - Paid access tiers

**Scoring Formula:**
- Total weight: 90 points
- Risk Score = (Present flags weight / Total weight) × 10
- Risk Level: 0-3 = LOW, 3-5 = MEDIUM, 5-7 = HIGH, 7-10 = CRITICAL

### Verification Levels

**5-Tier System:**
1. **Unverified** - Insufficient data
2. **Partially Verified** - Pattern matches, limited evidence
3. **Verified** - Confirmed scam (5+ victims)
4. **Highly Verified** - Multiple evidence sources
5. **Legitimate** - Verified safe account
