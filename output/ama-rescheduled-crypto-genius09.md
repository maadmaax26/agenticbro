## 🎯 Community Education Opportunity

**Teachable Moments:**

1. **Verified Accounts Can Be Scammers**
   - Even verified accounts with blue checkmarks can scam
   - Verification badge ≠ legitimacy
   - Always verify services before paying

2. **Service-Based Scams**
   - Paying for AMAs/giveaways is a common scam pattern
   - Always get refund policy in writing
   - Use escrow for larger amounts ($100+)

3. **Due Diligence Checklist**
   - Check for previous scam reports
   - Get references from other projects
   - Verify track record
   - Use escrow for large payments
   - Start with small test payments

4. **Red Flag Detection**
   - Unrealistic promises = red flag
   - Guarantees = red flag
   - Urgency/FOMO = red flag
   - Requests crypto = red flag
