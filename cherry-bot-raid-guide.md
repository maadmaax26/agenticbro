## 🎯 Setting Up Raids

### Basic Raid Setup

**Step 1: Initiate Raid Command**
```
/raid
```

This will:
- Lock the group (only admins can post)
- Show inline keyboard buttons for setup
- Guide you through configuration

**Step 2: Set Target Post URL**
- Paste the X post URL you want to raid
- Example: `https://x.com/username/status/123456789`

**Step 3: Set Target Metrics**

Choose engagement goals:
- **Likes:** Minimum likes target (e.g., 100)
- **Comments:** Minimum comments target (e.g., 50)
- **Retweets:** Minimum retweets target (e.g., 25)
- **Bookmarks:** Minimum bookmarks target (e.g., 10)

**Step 4: Optional: Set Bounty Amount**

**Minimum bounty amounts by blockchain:**
- SOL: 0.1 SOL
- BNB: 0.1 BNB
- ETH: 0.05 ETH
- TRX: 200 TRX

**Setting bounty benefits:**
- Reduces token supply (burn mechanism)
- Can positively impact $AGNTCBRO price
- Increases motivation for raid participation

**Step 5: Start Raid**
- Confirm all settings
- Cherry Bot starts tracking metrics
- Group remains locked until targets are achieved
- Auto-unlock when targets met
- If bounty set, token burning happens after unlock

### Post Selection Criteria

**Choose posts with:**
- ✅ 100+ retweets (high visibility)
- ✅ 500+ likes (active engagement)
- ✅ 50+ replies (active discussion)
- ✅ Relevant to crypto/DeFi/Solana
- ✅ From accounts with 10K+ followers
- ✅ Posted within last 24 hours (fresh content)

**Avoid posts with:**
- ❌ Obvious scam or rug pull content
- ❌ Unrealistic promises (x100, guaranteed returns)

**X rate limits:**
- 300 tweets per 3 hours
