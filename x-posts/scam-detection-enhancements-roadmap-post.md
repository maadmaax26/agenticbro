## 🎯 5 New Features We're Adding to the Roadmap

### 1. 🔥 VIP Tier Structure Detection
**Problem:** Scammers create multiple paid tiers (Tier 1: $100/mo → Tier 2: $500/mo → Scam)
**Solution:** Auto-detect multiple VIP tiers and flag unrealistic pricing
**Impact:** +7 points (HIGH RISK) automatically

### 2. 🏊 Pool Trading/Investment Pool Detection
**Problem:** "Pool trading investments" that never return funds
**Solution:** Scan for "pool trading", "investment pool", "collective investment" keywords
**Impact:** +8 points (HIGH RISK) - victims reported losing THOUSANDS

### 3. 🤖 Fake Engagement Detection
**Problem:** Bots, paid actors, fake testimonials make groups look legitimate
**Solution:** Analyze member count, message volume, like ratio, new member rate
**Impact:** Up to +10 points (CRITICAL RISK) when multiple red flags detected

### 4. 🎭 AI-Generated Content Detection
**Problem:** Deepfake AI videos, AI-written testimonials, voice cloning
**Solution:** Detect AI-generated text, deepfake videos/voices using advanced AI detection
**Impact:** +6 points (HIGH RISK) - emerging threat we're preparing for

### 5. 📘 Facebook Page/Group Scanning
**Problem:** Facebook fake pages, fake groups, fake ads, fake testimonials
**Solution:** Add Facebook page scanning, verify page legitimacy, detect fake testimonials
**Impact:** Up to +12 points (CRITICAL RISK) - massive platform being abused
