## Correct Raid Workflow

### Step 1: Find X Posts
Use `x.com/search-advanced` with filters:
- Keywords: "scam", "rug pull", "crypto safety"
- Min retweets: 100
- Min likes: 500
- Date: Last 24 hours

### Step 2: Evaluate Posts
- **Relevance:** Is it about scams/security?
- **Engagement:** Does it have good reach?
- **Opportunity:** Can AgenticBro add value?

### Step 3: Create Raid Target
Include:
- X post URL
- Author handle
- Engagement metrics
- Suggested comment
- Call to action (Like + Repost + Comment)

### Step 4: Post to Telegram
Post raid target to AgenticBro Telegram for community to raid
