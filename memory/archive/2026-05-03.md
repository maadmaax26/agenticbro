## Solana Ecosystem Enhancement Ideas (from X research)

Searched X/Twitter for capabilities the SOL ecosystem should have. Identified 11 enhancement opportunities for Agentic Bro:

### Critical (No competitors exist)
- **TOCTOU Attack Detector** — Detect state manipulation between simulation and signing
- **Durable Nonce Risk Scanner** — Scan wallets for dormant pre-signed transactions (Drift exploit used this)
- **Agentic Drainer Defense** — AI-vs-AI defense against malicious AI agents targeting wallets

### High Priority
- **Approval Crasher Detector** — Flag "Sign All" requests that bypass simulation errors
- **Honeypot 2.0 Scanner** — Detect hidden Tax Modifiers, Balance Overlays behind "renounced" mint authority
- **Account Poisoning / Dust Attack Detector** — Flag dust transactions and lookalike addresses, auto-close spam token accounts

### Medium Priority
- **Social Engineering Pattern Detector** — Cross-reference X profiles with on-chain behavior
- **Jito Bundle / MEV Risk Analyzer** — Detect bundled supply and fake volume at token launches
- **Wallet Permission Health Score** — Scan accumulated dApp approvals, suggest revocations
- **Cross-Chain Drainer Intelligence** — Pull threat intel from phish.report, Recordured Future, GoPlus
- **AI Agent Verification Service** — "Verified Agent" badge for Solana Agent Skills, certify safe agents

### Key Insight
Drift's $285M exploit (durable nonce + 6-month social engineering) validates the need for ALL of these tools. No consumer-facing tools exist yet for TOCTOU detection, durable nonce scanning, or agentic drainer defense — biggest greenfield opportunities.
