# Jeevs Mascot Design Specification

**Purpose:** Brand mascot for AgenticBro AI-powered scam detection system

---

## Character Design

### Core Concept
Jeevs is a friendly, trustworthy AI security guardian that protects users from scams. Think "friendly robot meets crypto security."

### Visual Elements

**Head:**
- Sleek, rounded helmet/face plate
- Green LED visor that "scans" (horizontal light bar)
- Calm, intelligent expression
- Small antenna with green tip

**Body:**
- Compact, rounded robot torso
- Shield emblem on chest with "scan" icon
- Green accent lighting along edges
- Solana purple subtle glow on sides

**Accessories:**
- Small Mac Studio server node as "backpack" or base
- Scanning laser pointer in one hand
- Shield held in other hand

**Colors:**
- Primary: White/light gray (#FFFFFF, #F5F5F5)
- Accent: Agentic Green (#16A34A)
- Secondary: Solana Purple (#9945FF) - subtle
- Alert: Red (#EF4444) - only for scam warnings

### Personality Traits
- Protective but approachable
- Always scanning, always vigilant
- Trustworthy guardian
- Fast, efficient, intelligent

---

## Usage Variations

### 1. Hero Image (Website)
- Full body character standing confidently
- Scanning a profile/token with visor
- Shield raised in protection pose
- Background: Gradient white to light green

### 2. Telegram Avatar
- Head/shoulders only
- Visor glowing green
- Shield icon visible
- Circular crop ready

### 3. Pitch Deck
- Professional pose with Mac Studio node
- "Scanning" animation pose
- Shield prominently displayed

### 4. Scam Warning
- Alert pose with red accent
- Shield blocking danger
- "Warning" visual elements
